# PR Self-Review Checklist

Before opening this pull request, I confirm the following:

- [ ] The code does what the PR title and description claim.
- [ ] All tests pass locally and new tests were added if needed.
- [ ] The README is updated and reflects any changes in setup or usage.
- [ ] This PR focuses on a single responsibility and does not mix unrelated changes.
- [ ] There are no debug artifacts (no print statements, no breakpoint(), no commented test code).
- [ ] Commit messages are clear, descriptive, and follow the expected format.
- [ ] I reviewed the diff carefully before submitting.
